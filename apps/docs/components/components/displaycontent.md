# Display Content

Display Content component provide title, subtitle and description for your banners. It can be wrapped with Display or Hero component to position the content inside. 

<Generate />

| title           | `string`    |               |                                   |
| subtitle        | `string`    |               |                                   |
| description     | `string`    |               |                                   |
<!-- react -->
| className       | `string`                |               |                                   |
| children        | `ReactNode`             |               |                                   |
